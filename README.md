題目說明
---
利用分組法刪除等價項目，化簡狀態表↓
1. 根據 input 的輸出狀態分組 
2. 根據同 group 的各狀態之次態是否在同組 
→同組：留著 / 不同：拆組（次態在同組者分到同⼀組） 
3. 重複動作2直到無法再分組 
4. 合併，狀態表只留下各組排頭，同組者由排頭替代
